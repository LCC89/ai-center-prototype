I 多模态

POST /v1/chat/completions

接口说明

对话生成接口，支持多轮对话和流式响应。通过该接口可以与平台接入的各种大模型进行交互，支持设置系统提示词、温度参数、最大token数等。

可用模型

模型名称 模型ID API URL 上下文长度
Qwen3-14B Qwen/Qwen3-14B http://220.196.22.85:8000/v1 128K tokens
GLM-4.1V-9B-Thinking THUDM/GLM-4.1V-9B-Thinking http://220.196.22.85:8001/v1 64K tokens
DeepSeek-R1-Distill-Llama-70B deepseek-ai/DeepSeek-R1-Distill-Llama-70B http://220.196.22.85:8002/v1 128K tokens

请求示例 (Qwen3-14B)

curl 
curl http://220.196.22.85:8000/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "Qwen/Qwen3-14B",
    "messages": [
      {"role": "system", "content": "你是一个 helpful 的 AI 助手"},
      {"role": "user", "content": "你好，请介绍一下自己"}
    ],
    "temperature": 0.7,
    "max_tokens": 2000,
    "stream": false
  }'
Python 
import requests

response = requests.post(
    "http://220.196.22.85:8000/v1/chat/completions",
    headers={
        "Content-Type": "application/json",
        "Authorization": "Bearer YOUR_API_KEY"
    },
    json={
        "model": "Qwen/Qwen3-14B",
        "messages": [
            {"role": "system", "content": "你是一个 helpful 的 AI 助手"},
            {"role": "user", "content": "你好，请介绍一下自己"}
        ],
        "temperature": 0.7,
        "max_tokens": 2000
    }
)

print(response.json())

请求参数

参数名 类型 必填 说明
model string 是 模型ID，如 Qwen/Qwen3-14B、ZhipuAI/GLM-4.1V-9B-Thinking 等
messages array 是 对话消息列表，包含 role 和 content
temperature number 否 采样温度，范围 0-2，默认 0.7
max_tokens integer 否 最大生成token数，默认 2000
stream boolean 否 是否使用流式响应，默认 false
